# Proof of Human Work

A short talk about asymmetric effort, slop, and what it means for a human to leave their fingerprints on the work.

## Viewing the slides

Open `slides.html` in any modern browser.

- `→` / `Space` — next slide
- `←` — previous slide
- Click the right/left half of the screen to navigate

## Files

- `slides.html` — self-contained HTML/CSS/JS slideshow
- `proof-of-human-work.md` — source outline
- `revs/n-12.md` … `revs/n-0.md` — draft history shown in the animated slide
- `*.png` — slide imagery
